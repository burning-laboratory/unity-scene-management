# Scenes Group

Asset with scene group settings.

### Settings:
- **-** **`Group Name (string)`** - Scenes group name.

- **-** **`Scenes (List<SceneData>)`** - Included scenes in scene group.

### Public properties:
- **-** **Group Name (string)`** - Scenes group name.

- **-** **`Scenes (List<SceneData>)`** - Included scenes in scene group.
